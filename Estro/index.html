Estro.js

Implements needed and often used string functionality directly into strings.

Never overrides any native or custom implementations; it only adds functionality.

Extended String Object

Usage

Include the estro.js source file before any JavaScript in which you wish to use the Prototype methods.

Methods

  • String.compareTo Compare one string to another

  • String.equals Check one string against another.

  • String.equalsIgnoreCase Check one string against another without case sensitivity.

  • String.startsWith Check to see if a string starts with the given string.

  • String.endsWith Check to see if a string ends with the given string.

  • String.contains Check to see if a string contains a given string.

  • String.trim Trim the string of whitespace. Returns: String

  • String.trimLeft Trim the left of the string of whitespace. Returns: String

  • String.trimRight Trim the right of the string of whitespace. Returns: String

  • String.isEmpty Check to see if the string is empty, and really empty.

  • String.isBlank Check to see if the string is null, undefined, blank.

  • String.reverse Reverses string.

  • String.repeat Repeat string n times

  • String.capitalize Capitalize each word in the string and lower the rest.

  • String.camelize CamelCase strings.

  • String.dasherize Hyphenate Strings.

  • String.underscore Underscore strings.

  • String.bind Small template layer / printf.

  • String.extract Extract data into an array based on regex and group.

  • String.toInt Convert string into numeric value.

  • String.toHash Converts the string to a hashcode. Based on Java.
